## 优化

> 相比原版 trojan 我们优化的部分

| 修改后 | 原版 |
| :--- | :--- |
| 用户授权缓存并一分钟无阻塞更新 | 无缓存，同一用户不同请求均验证，服务器与数据库校验过慢 |
| 流量数据缓存并一分钟上传一次流量 | 无缓存，每个请求结束后都上传一次流量 |
| 审计规则支持并三十分钟刷新一次审计（被拦截的审计一分钟上传一次 | 无支持 |
| 上传当前在线用户数量 | 无支持 |
| 可查看累计链接用户数量 | 无支持 |
